# canvas-m8

Shared canvas for remote pair programming.

## What it is

canvas-m8 is a lightweight web app that provides a shared drawing canvas for remote pair programming, architecture sketching, and whiteboarding.

## Getting started

### Prerequisites

- Node.js (LTS recommended)
- npm

### Install

```sh
npm install
```

### Run locally

This repo has a Vite frontend and a small Node server.

In one terminal:

```sh
npm run dev
```

In another terminal:

```sh
npm run dev:server
```

Open the URL printed by Vite.

### Build

```sh
npm run build
```

### Production

```sh
npm start
```

## CLI

This project also exposes a small CLI named `m8` (see `bin/m8.js`).
